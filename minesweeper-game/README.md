# 5x5 扫雷游戏

一个使用Python 3.11开发的经典扫雷游戏，具有良好的跨平台兼容性，支持Windows、Linux和macOS系统。提供多种运行方式，包括命令行版本和Web版本，满足不同用户的需求。

## 🎮 游戏特点

- **经典玩法**：5x5网格，包含5个地雷
- **多彩界面**：彩色命令行版本，数字用不同颜色标识
- **智能保护**：第一次点击保护，确保首次点击及周围没有地雷
- **自动展开**：空白区域自动展开，提升游戏体验
- **标记功能**：支持标记/取消标记地雷
- **状态显示**：实时显示剩余地雷数和游戏状态
- **重新开始功能**：随时重新开始新游戏
- **跨平台兼容**：Windows/Linux/macOS全平台支持

## 📦 包含版本

1. **彩色命令行版本** (`src/minesweeper_color.py`) - 推荐使用
   - 彩色显示，数字不同颜色
   - 跨平台兼容，各系统优化
   - 命令行界面，保留原版体验

2. **基础命令行版本** (`src/minesweeper.py`)
   - 无颜色显示，纯字符界面
   - 兼容性最好，适合老旧终端

3. **Web版本** (`web/minesweeper_web.html`)
   - 图形界面，鼠标操作
   - 无需安装Python，浏览器直接运行
   - 支持计时器和难度选择

## 🚀 快速开始

### 方法一：直接运行Python文件
1. 确保已安装Python 3.11或更高版本
2. 在命令行中运行：

   ```bash
   # 基础版本
   python3 src/minesweeper.py
   
   # 彩色版本（推荐）
   python3 src/minesweeper_color.py
   ```

### 方法二：Web版本
直接用浏览器打开 `web/minesweeper_web.html` 文件即可运行

### 方法三：Windows用户专用
Windows用户可以参考 [Windows特定指南](docs/windows-guide.md) 获取更多便捷的启动方式，包括批处理文件和可执行文件。

## 🎯 游戏规则

### 基本操作
- **揭开格子**：输入坐标，格式为"行列"（例如：12表示第1行第2列）
- **标记地雷**：输入"f+坐标"（例如：f12标记第1行第2列）
- **重新开始**：输入"r"
- **退出游戏**：输入"q"

### 获胜条件
- 揭开所有非地雷格子即可获胜
- 数字表示周围8个格子中地雷的数量
- 合理使用标记功能帮助记忆地雷位置

## 🐛 故障排除

### 常见问题

**Q: 运行时显示中文乱码**
A: 确保终端支持UTF-8编码，或使用Windows用户专用的批处理文件启动

**Q: 颜色显示不正常**
A: 尝试使用基础版本，或在现代终端中运行

**Q: Python未安装**
A: 从 https://www.python.org/downloads/ 下载安装Python 3.11+

**Q: 无法生成exe文件**
A: 确保网络连接正常，以便安装PyInstaller依赖
